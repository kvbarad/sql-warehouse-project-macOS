#**Getting Started**

Prerequisites
Before you begin, ensure you have the following installed:
- macOS Monterey or newer
- Docker Desktop for Mac
- Visual Studio Code
- Bash shell (standard on macOS)
- (Recommended) SQL Server extension for VSCode
